Подключение и вход в веб-интерфейс

  • Подключитесь к роутеру: по кабелю в порт LAN1, или к заводской Wi-Fi сети (имя/пароль на наклейке)
  • Откройте браузер и зайдите на 192.168.100.1 или 192.168.1.1 в зависимости модели (точный адрес - на наклейке/в инструкции)

Если не заходит в веб-интерфейс:

  • Перезагрузите модем отключив по питанию на минуту (некоторые модемы имеют защиту, и блокируют вход в админку после нескольких часов работы)
  • Проверьте что ваше устройство получило IP адрес из той же подсети что и сеть модема 192.168.1.x либо 192.168.100.x

Настройка сетевой карты в системе Windows 10 при автоматических настройках подключения (DHCP).
  1. Откройте меню Пуск (Start) - Панель управления(Control Panel) - Сеть и Интернет(Network and Internet) (если этот раздел отсутствует, см. следующий) - Центр управления сетями и общим доступом(Network and Sharing Center) - Изменение параметров адаптера(Change adapter settings) (в левой части окна).
  2. Щелкните правой кнопкой мыши на ярлык Подключение по локальной сети (Local Area Connection) и выберите Свойства (Properties).
  3. В появившемся окне выделите пункт Протокол Интернета версии 4 (TCP/IPv4) (Internet Protocol version 4 (TCP/IPv4)) и нажмите левой кнопкой мыши по кнопке Свойства (Properties).
  4. Выберите пункт Использовать следующий IP-адрес (Use the following IP address). В строке IP-адрес (IP address) впишите значение (без кавычек) 192.168.1.XX (вместо XX число от 10 до 254), в строке Маска подсети (Subnet mask)– значение 255.255.255.0, в строке Основной шлюз (Default gateway) – 192.168.1.1 либо 192.168.100.1 в зависимости от роутера
    5. Выберите пункт Использовать следующие адреса DNS-серверов (Use the following DNS server addresses и введите в строке Предпочитаемый DNS-сервер (Preferred DNS server) 194.158.196.245 и Альтернативный DNS-сервер (Alternate DNS server) значения 86.57.255.149. Далее нажмите ОК
  5. Нажмите Закрыть (Close).
Возможные нюансы:
  • В некоторых случаях настройки принимаются только после презагрузки
  • Если при настройке сети требует указать префикс подсети - указывать /24
  • Не рекомендуется отключать - "Протокол Интернета версии 6 (TCP/IPv6)" т.к. используется для внутренних служб Windows
  • "Конфигурация с несколькими основными шлюзами" - появляется если несколько интерфейсов настроено с одним шлюзом (Например Wi-Fi и Ethernet имеют одинаковый шлюз)
  • Если настройки сетевой слетают необходимо проверить/обновить сетевые драйвера, сделать сброс сети

Универсальная инструкция для большинства роутеров (TP-Link, Keenetic, ASUS, Tenda и другие). Подходит для любого роутера с функцией подключения по PPPoE - пункты меню могут отличаться, ориентируйтесь на смысл.

Дополнительные параметры которы по-умолчанию не запрашивает но могут понадобиться для расширенной настройки:
  • MTU: 1492 (по умолчанию)
  • Type: PPPoE client
  • Поле Service Name - оставить пустым
  • Use Peer DNS - Enable
  • Add default route - Enable
  • Authentication type - PAP/CHAP (Поддерживает оба режима)

Возможные проблемы:
  • PPPoE не подключается: проверьте логин/пароль, MTU 1492, кабель в WAN, перезапустите ONT/модем и роутер, убедитесь что модем переведен в bridge
  • PPPoE подключается но выдает странный IP либо заглушку - неверный логин/пароль
  • PPPoE подключается и сразу отваливается: обновите прошивку, проверьте кабель/питание, отключите сомнительные фильтры, проверьте логи на вашем оборудовании

Параметры антенны

Для приема эфирного цифрового телевидения нужна дециметровая антенна, работающая в диапазоне частот 470-862 МГц. Рекомендуемые характеристики активной (комнатной) и пассивной (наружной) телевизионной антенны:

  • частотный диапазон ДМВ 470 МГц (21…69 каналы);
  • коэффициент усиления 18 дБ и 15 дБ соответственно.

Нецелесообразно применять антенны следующих типов:

  • антенны метрового диапазона волн, так как для вещания цифрового эфирного телевидения в Республике Беларусь применяется только дециметровый диапазон.
  • «польские» антенны. При использовании антенны типа «польская» с усилителем не гарантируется качественный прием цифрового сигнала.
  • старые антенные системы могут в своем составе содержать элементы (сама антенна, радиокабель, сплиттеры, разъемы и т.д.), работа которых отрицательным образом может сказываться на качестве приема цифрового сигнала.

В продаже имеются антенны как со встроенным усилителем (активные), так и без него (пассивные). Активные антенны для питания усилителя требуют подключения к электросети или активизации в цифровом приемнике функции питания антенны.

Чем больше расстояние до передающей станции, тем с большим коэффициентом усиления должна быть антенна. Для приема близлежащих мощных станций (расстояния до 20-30 км) достаточно антенны с усилением 12-15 дБ.

  • Далее выбрать внешнюю авторизацию (вход с помощью сайта в приложении) - Вас перенаправит на авторизацию в «Мой Белтелеком»
