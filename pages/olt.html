Информация о ONT

Подробная информация о ONT
(config-if-gpon-0/9)#display ont info 10 23
Статистика отключений
(config-if-gpon-0/9)#display ont register info 6 8
Список MAC
display mac-address port 0/5/4 ont 9
Статус Ethernet портов
(config-if-gpon-0/9)#display ont port state 1 13 eth-port all
Трафик
(config-if-gpon-0/2)#display ont traffic 9 12
Проверить прошивку
display ont version 11 1
Оптические параметры ONT
(config-if-gpon-0/4)#display ont optical-info 9 2
Настройки WAN на ONT (Маршрутизация)
(config-if-gpon-0/4)#display ont wan-info 10 23
Порт-маппинг по таблице роутинга
(config-if-gpon-0/13)#display ont routing-table policy-route 11 2

Wi-fi

Статус WLAN
(config-if-gpon-0/13)#display ont wlan-info 11 2
Состояние каналов wi-fi и кол-во подключенных
(config-if-gpon-0/13)#display ont wlan-status 11 2

Информация о статусе ONT по блоку

Состояние абонентов по сплиттеру
display ont info summary 0/1/7
Состояние абонентов по сплиттеру (статусы)
display ont info 0 16 3 all
Оптические параметры по всему блоку
(config-if-gpon-0/5)#display ont optical-info 14 all

Пинг и ARP

Пинг с OLT
ping 10.165.107.163 ping -c 10 10.164.197.139 ,где -c 10 кол-во пакетов
Пинг с модема
(config-if-gpon-0/1)#ont remote-ping 0 4 ip-address 10.10.8.8
ARP
display arp all либо display arp 10.165.107.163
Трассировка с OLT
tracert -q 5 10.10.0.10 tracert ipv6 -q 5 2001:db8:100::3

VoIP (POTS)

Проверить состояние POTS(FXS) портов
(config-if-gpon-0/9)#display ont port state 1 13 pots-port all
Проверить статус регистрации IMS по сплиттеру
display ont pots state 0/14/2

Поиcк

Поиск по серийному (SN)
display ont info by-sn 44504450AAAAAAAA
Поиск незарегистрированных
display ont autofind all
Поиск по определенному MAC
display location 4cf9-5dd3-1aaa

Регистрация

Разрегистрация ONT
interface gpon 0/1 (config-if-gpon-0/12)#ont delete 3 4
Регистрация ONT
interface gpon 0/0 (config-if-gpon-0/0)#ont add 2 3 sn-auth 4857544AAAAAAA omci ont-lineprofile-id 2001 ont-srvprofile-id 1001 desc "ONT_NO_DESCRIPTION"
Смена ONT без перерегистрации (Короткая)
interface gpon 0/0 ont modify 'port' 'ont-id' sn 'new-sn'

Сервисы

Список сервисов на порту
display service-port port 0/1/2 ont 3
Отключить сервис по индексу
undo service-port 578
Информация о сервисе по индексу
display service-port 11348
Отключить "зависший" сервис
btv igmp user delete service-port 578 quit undo service-port 578
Отключить DHCP
(config-if-gpon-0/1)#ont ipconfig 0 4 dhcp reset

Активировать порт

Активировать/деактивировать ONT на порту
(config-interface-gpon-0/0)#ont activate 1 2 - Активировать (config-interface-gpon-0/0)#ont deactivate 1 2 - Деактивировать

Reset и Reboot

Перезагрузка ONT
(config-if-gpon-0/13)#ont reset 2 1
Ресет (Полный сброс)
(config-if-gpon-0/13)#ont factory-setting-restore 2 1

Мультикаст

Проверить Multicast
display service-port port 0/14/13 ont 2 display igmp user service-port 11911
Добавить Multicast
btv (config-btv)#igmp user add service-port 642 no-auth
Прописать мультикаст на 4 STB
btv (config-btv)#igmp user modify service-port 642 max-program 4
Удалить Multicast
btv (config-btv)#igmp user delete service-port 642

Настроить порт

Настроить порт на INET
conf t (config)#pon-onu-mng gpon-onu_1/7/4:18 (gpon-onu-mng)#vlan port eth_0/4 mode tag vlan 10 end
Настроить порт на IPTV + multicast
conf t (config)#pon-onu-mng gpon-onu_1/7/4:18 (gpon-onu-mng)#vlan port eth_0/4 mode tag vlan 20 либо (gpon-onu-mng)#vlan port eth_0/4 vlan 1000 (gpon-onu-mng)#vlan port eth_0/4 mode hybrid def-vlan 1000 def-prio 4 (gpon-onu-mng)#dhcp-ip ethuni eth_0/4 from-internet (gpon-onu-mng)#multicast vlan tag-strip port eth_0/4 enable end
Удалить услуги с 4-го порта
conf t (config)#pon-onu-mng gpon-onu_1/7/4:18 no vlan port eth_0/4 mode no vlan port eth_0/4 vlan 1000 end

DHCP

Снять DHCP с 4-го порта
conf t (config)#pon-onu-mng gpon-onu_1/7/4:18 dhcp-ip ethuni eth_0/4 from-internet end
Вернуть DHCP на 4-й порт
conf t (config)#pon-onu-mng gpon-onu_1/7/4:18 dhcp-ip ethuni eth_0/4 from-onu end

VoIP (POTS)

Статус регистрации IMS
show gpon remote-onu voip-linestatus gpon-onu_1/9/3:18
Настроить телефонию на порт
conf t pon-onu-mng gpon-onu_1/7/4:20 no sip-user pots_0/1 sip-user pots_0/1 sip-agent 2 user-aor +375172421200 username +375172421200@ims.beltel.by password 12345678

либо второй вариант:

sip-service pots_0/1 profile VOIP-SIP1 userid +375172421202 username +375172421202@ims.beltel.by password 12345678
Снять телефонию
conf t pon-onu-mng gpon-onu_1/7/4:20 no sip-user pots_0/1 no sip-service pots_0/1 - если не отработал первый вариант

Поиcк

Поиск по серийному (SN)
show gpon onu by sn ZTEGC1B3AAAA
Поиск незарегистрированных
show gpon onu uncfg
Поиск по MAC
show mac 9813.33f8.AAAA

Регистрация

Разрегистрация ONU
conf t interface gpon-olt_1/12/7 (config-if)#no onu 7
Регистрация ONU
conf t interface gpon-olt_1/2/2 (config-if)onu 17 type F670L sn ZTEGC11F4854 exit
Смена ONU без перерегистрации (Короткая)
conf t interface gpon-onu_x/y/z:a (config-if)#registration-method sn ZTEGBxxxxx exit

Режимы работы

router
conf t (config)#pon-onu-mng gpon-onu_1/6/7:25 (gpon-onu-mng)#switchport-bind switch_0/1 iphost 1 (gpon-onu-mng)#pppoe 1 nat enable user 176000999990001@beltel.by password 123123123 (gpon-onu-mng)#vlan-filter-mode iphost 1 tag-filter prio-vid-filter untag-filter discard (gpon-onu-mng)#vlan-filter iphost 1 priority 0 vid 3114
bridge
conf t (config)#pon-onu-mng gpon-onu_1/6/7:25 (gpon-onu-mng)#no switchport-bind iphost 1 (gpon-onu-mng)#no pppoe 1 (gpon-onu-mng)#no vlan-filter-mode iphost 1 (gpon-onu-mng)#no vlan-filter iphost 1 priority 0 vid 3114

Сервисы

Снять PPPoE
conf t pon-onu-mng gpon-onu_1/6/7:25 (gpon-onu-mng)#pon-on-mng (gpon-onu-mng)#no pppoe 1
Снять IMS
conf t pon-onu-mng gpon-onu_1/3/1:1 no sip-user pots_0/1 no sip-service pots_0/1 - если первый вариант не отработал
Снять IPTV
conf t no vlan port eth_0/2 mode no vlan port eth_0/2 vlan 1000

Активировать ONT на порту

Активировать
conf t (config)#interface gpon-onu_1/8/7:1 (config-if)#no shutdown
Деактивировать (отключить порт)
conf t (config)#interface gpon-onu_1/8/7:1 (config-if)#shutdown

Reset и Reboot

Перезагрузка
conf t (config)# pon-onu-mng gpon-onu_1/1/8:5 (gpon-onu-mng)# reboot
Cброс ONT на заводские (Ресет)
conf t (config)# pon-onu-mng gpon-onu_1/1/8:5 (gpon-onu-mng)# restore factory

Multicast

Проверить мультикаст
show gpon remote-onu multicast vlan gpon-onu_1/9/3:18 либо show igmp mvlan 1000 и искать в списке
Добавить мультикаст
conf t (config)#igmp mvlan 1000 receive-port gpon-onu_1/4/5:1 vport 2
Удалить мультикаст
conf t (config)#no igmp mvlan 1000 receive-port gpon-onu_1/4/5:1 vport 2

TR-69

Прописать управление по TR
conf t (config)#pon-onu-mng gpon-onu_1/7/4:18 (gpon-onu-mng)veip 1 port udp 1000 host 3 (gpon-onu-mng)reboot end
